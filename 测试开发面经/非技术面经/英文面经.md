### 1、how to report a bug?
to report a bug, we need create a bug card, which include basic element：
card number, card status, card priority,card title
some label to classify the bug: if backend or frontend, if business problem or techinical error, 
body:
	environment
	used role account
	the steps to reproduce
	the real result and expactation
	the root cause
after report the bug and transfer it to devs, dev will fix it, then the associated functions will be retested. if all ok, the bug card will be closed.

### 算法题：最长不重复子串
以下是使用滑动窗口算法解决“最长不重复子串”问题的 Python 代码示例：
```python
s=input().strip()
n=len(s)
def longth_longest_substring(s):
    ans=0
    left=0
    char_index={}
    for right in range(n):
        if s[right] in char_index and left<=char_index[s[right]]:
            left=char_index[s[right]]+1
        else:
            ans=max(ans,right-left+1)
        char_index[s[right]]=right
    return ans
print(longth_longest_substring(s))
```
### 找出最短的数字子串
```python
def shortest_substring(s, target_chars):
    target_set = set(target_chars)  # 将目标字符转换为集合，方便快速查找
    min_length = float('inf')  # 初始最小长度为正无穷
    left = 0  # 左指针
    right = 0  # 右指针
    current_set = set()  # 当前窗口中的字符集合

    while right < len(s):
        current_set.add(s[right])  # 将右指针指向的字符添加到当前集合
        right += 1

        while current_set.issuperset(target_set):  # 如果当前集合包含所有目标字符
            min_length = min(min_length, right - left)  # 更新最小长度
            current_set.remove(s[left])  # 从当前集合中移除左指针指向的字符
            left += 1

    return min_length if min_length!= float('inf') else 0  # 如果找到最短子串则返回长度，否则返回 0

s = "bvcabcdeffghijklmmnopqrstuvwxzyzz"
target_chars = "abcdefghijklmnopqrstuvwxyz"
print(shortest_substring(s, target_chars))  
```

### 找出最长的连续数字子串
```python
import sys
s=sys.stdin.readline()
def longest_string(s):
    max_str=''
    for left in range(len(s)):
        current_str=''
        for right in range(left,len(s)):
            if s[right].isdigit():
                current_str=current_str+s[right]
            else:
                break
        if len(max_str)<len(current_str):
            max_str=current_str
    return max_str
print(longest_string(s))
```
**设计思路**：
1、通过外层的循环(loop structure)遍历(go through)字符串 `s` 的每个位置作为起始位置 left`
2、构建内层循环，右指针right从左指针left位置开始移动
3、初始化一个缓存子串current_str，当右指针遇到数字时，将数字添加到current_str，如果不是数字，就停止当前的内层循环。
4、每次构建完一个 `current_str` 后，将其长度与 `max_str` 的长度进行比较，如果 `current_str` 更长，就更新 `max_str` 
**知识点**：
跳出本次循环使用 `continue` 关键字，结束整个循环使用 `break` 关键字。


```python
import sys
s=sys.stdin.readline()

def longest_string(s):
    left=0
    right=0
    max_str=''
    tmp_str=''
    while right <len(s):
        if s[right].isdigit():
            tmp_str+=s[right]
            right+=1
        else:
            left=right+1
            right+=1
            if len(max_str)<len(tmp_str):
                max_str=tmp_str
            tmp_str=''
    return max_str
print(longest_string(s))
```
**设计思路**
滑动窗口方法 ，定义了left和right两个指针，max_str存储最长的连续数字子串，tmp_str临时存储当前窗口内的数字子串
进入while循环，只要right指针不超过字符串范围，就进行以下操作：
当s[right]是数字时，将其添加到tmp_str中，right指针右移一位。
当s[right]不是数字时，左指针移动到right+1，准备开始新的窗口，右指针后移一位。比较max_str和right_str的长度，如果 `tmp_str` 更长，就更新 `max_str` 为 `tmp_str` 的值，并将 `tmp_str` 清空，为下一次寻找做准备。
当while循环结束，返回存储在max_str的最长子串。
**知识点**
#滑动窗口 避免重复遍历，定义左右指针，注意窗口移动时左右指针都要移动

        while current_node.next:
            current_node=current_node.next
        current_node.next=new_node

        current_node=self.head
        while current_node:

            current_node=current_node.next

### 绘图机器
#### 题目描述
**绘图机器**的绘图笔初始位置在原点(0,0)机器启动后按照以下规则来进行绘制直线。
1. 尝试沿着横线坐标正向绘制直线直到给定的终点E
2. 期间可以通过指令在纵坐标轴方向进行偏移，offsetY为正数表示正向偏移,为负数表示负向偏移
给定的横坐标终点值E 以及若干条绘制指令，
请计算绘制的直线和横坐标轴以及x=E的直线组成的图形面积。
#### 输入描述

- 首行为两个整数 N 和 E
- 表示有N条指令,机器运行的横坐标终点值E
- 接下来N行 每行两个整数表示一条绘制指令**x offsetY**
- 用例保证横坐标x以递增排序的方式出现
- 且不会出现相同横坐标x
取值范围
- 0<N<=10000
- 0<=x<=E<=20000
- -10000<=offsetY<=10000

#### 输出描述
- 一个整数表示计算得到的面积 用例保证结果范围在0到4294967295之内。
#### 用例

|---|---|
|输入|4 10<br><br>1 1<br><br>2 1<br><br>3 1<br><br>4 -2|
|输出|12|
|说明|无|

|     |                            |
| --- | -------------------------- |
| 输入  | 2 4<br><br>0 1<br><br>2 -2 |
| 输出  | 4                          |
| 说明  | 无                          |
```python
n,end_x=map(int,input().split())
x_y_matrix=[list(map(int,input().split())) for _ in range(n)]

def caiculate_area(n,end_x,matrix):
    area=0
    previous_x=0
    previous_y=0
    for i in range(n):
        current_x=x_y_matrix[i][0]
        current_y=x_y_matrix[i][1]
        area+=(current_x-previous_x)*abs(previous_y)
        previous_x=current_x
        previous_y+=current_y
    if x_y_matrix[n-1][0]<end_x:
        area+=(end_x-x_y_matrix[n-1][0])*abs(previous_y)
    return area
print(caiculate_area(n,end_x,x_y_matrix))
```

### 删除有序数组的重复项
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        tmp_nums=[]
        numbers_account=0
        for i in range(len(nums)):
            if nums[i] not in tmp_nums:
                tmp_nums.append(nums[i])
                numbers_account+=1
        nums[:]=tmp_nums#切片赋值
        return numbers_account
```
#切片赋值 ```nums[:] = tmp_nums``` 是直接修改```nums```的元素，引用的对象没变

#直接赋值 ```nums = tmp_nums``` `nums` 不再指向原来的列表对象，而是直接指向了 `tmp_nums` 所指向的列表对象

#### 题目解析

首先，我们需要理解题目的输入和输出要求。题目提供了两行输入，第一行是多个用逗号分隔的人名，第二行是待查找的字符串。我们的目标是找出那些人名，其每个单词的连续前几位能够组成给定的查找字符串，并且必须用到每个单词的至少一部分。

接下来，我们可以按照以下步骤进行解题：

1. 读取并分割人名：读取第一行输入，使用逗号作为分隔符，将人名分割成一个列表或数组。
2. 遍历人名列表：对于人名列表中的每一个单词，我们需要进行查找和匹配操作。
3. 查找和匹配：对于每个人名单词，我们需要检查它是否包含在给定的查找字符串中，并且这个单词的起始位置之前和之后都不能有其他字符（即该单词必须是查找字符串的连续部分）。
4. 输出匹配的人名：如果某个单词满足上述条件，即它是查找字符串的连续部分，并且没有与其他单词重叠，我们就将其输出。需要注意的是，由于题目要求用到每个单词，因此我们需要确保所有单词都被匹配到，并且每个单词只被使用一次。
5. 处理特殊情况：如果某个单词在查找字符串中出现了多次，我们需要确保每次匹配都是独立的，即不会与其他单词的匹配产生重叠。
6. 输出结果：最后，将所有满足条件的人名按照题目要求的格式输出。
